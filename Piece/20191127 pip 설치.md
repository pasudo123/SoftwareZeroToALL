# PIP 설치
- 윈도우10 에서 PIP 설치 (64비트)

## 선행
- Python 2.7 설치. (msi 파일 다운 및 C:/Python27)
- Wget 설치. (wget.exe 파일을 환경변수 경로에 위치)
- https://bootstrap.pypa.io/get-pip.py 파일을 wget 명령어로 설치
  - 해당위치에서 ```python get-pip.py``` 명령어 수행
