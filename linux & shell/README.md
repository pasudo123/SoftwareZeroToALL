## <a id="nohup"></a> nohup 🔔
* HUP (HangUP) Signal 을 무시하는 명령어이다.
* HUP 이란 kill 명령어의 시그널 번호 `1`, `SIGHUP` 을 의미한다.
* 출력은 따로 리디렉션이 되지 않은 경우, nohup.out 이라고 불리는 파일에 쓰여진다.
